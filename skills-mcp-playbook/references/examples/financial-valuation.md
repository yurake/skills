# 財務バリュエーション（Comparable Company Analysis）例

- 目的: 複数データソースからライブ指標を取得し、一貫した手法でバリュエーションを生成。
- MCP接続: S&P Capital IQ / Daloopa / Morningstar など（ライブ市場データ取得）。
- Skillで書くこと:
  1. データソースの優先順位とクエリ対象（ティッカー/期間）
  2. 取得後の整形手順（指標揃え、通貨/期間調整）
  3. 評価手法の手順とチェック（倍率計算、アウトライヤ除外基準）
  4. 出力フォーマット（表構成、要約セクション、前提条件の明示）
  5. コンプラ確認ステップ（必須開示、免責）
- 衝突回避: MCPは生データ(JSON等)返却に寄せ、整形・表現はSkill側で統一。
- 簡易シーケンス: データ取得→整形→計算→検証→フォーマット→出力。
